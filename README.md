# 🚀 AR Projects (Gesture + Visual FX)

An experimental collection of **browser-based Augmented Reality projects** using **hand tracking (MediaPipe)** to create interactive experiences.

This repo focuses on:

* 🖐️ Gesture-based controls
* 🌌 Real-time visual effects
* 🎯 Camera-based interaction (no external hardware)

---

## 🧠 Projects Included

### 🖐️ Finger Launcher AR

Control your browser using hand gestures.

**How it works:**

* Shows number of fingers → triggers actions
* Opens websites based on gesture

**Gestures:**

* ☝️ 1 finger → YouTube
* ✌️ 2 fingers → LeetCode
* 🤟 3 fingers → ChatGPT
* 🖖 4 fingers → Instagram
* 🖐️ 5 fingers → Google

---

### 🌌 Galaxy Hand FX

A visual AR experience where your hands generate energy effects.

**Features:**

* 🔵 Real-time hand tracking
* ✨ Energy orb + galaxy effects
* 🤲 Dual-hand interaction
* 🔥 Smooth fade-in/out animations
* 🎨 Cinematic overlay

---

## ⚙️ Tech Stack

* **MediaPipe Hands**
* **JavaScript (Vanilla)**
* **HTML5 Canvas**
* **Web Camera API**

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/GarvChopra/Ar-projects.git
cd Ar-projects
```

### 2. Run locally

Just open the HTML files in browser:

```bash
open index.html
```

or use Live Server (recommended)

---

## ⚠️ Requirements

* Camera access enabled
* Modern browser (Chrome recommended)
* Good lighting for hand tracking
